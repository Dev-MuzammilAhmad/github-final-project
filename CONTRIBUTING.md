# Contributing to Simple Interest Calculator

We welcome contributions to this project! All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.

## How to Contribute

### Steps to Contribute
1. Fork the repository
2. Create a new branch for your feature or bug fix
3. Make your changes
4. Test your changes
5. Submit a pull request

### Reporting Bugs
If you find a bug, please open an issue with:
- A clear description of the bug
- Steps to reproduce it
- Expected vs actual behavior

### Suggesting Enhancements
We welcome ideas for enhancements! Please open an issue with:
- A clear description of the enhancement
- Why it would be useful
- Any implementation ideas you have

## Code of Conduct

Please note that this project is released with a Contributor Code of Conduct. By participating in this project you agree to abide by its terms.
